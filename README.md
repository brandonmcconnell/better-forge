# ▲ / better-forge

**Production-grade Turborepo template for Next.js apps.**

Based on Vercel's [next-forge](https://github.com/vercel/next-forge) template, originally created by [Hayden Bleasel](https://github.com/haydenbleasel).

## Overview

better-forge is a [Next.js](https://nextjs.org/) project boilerplate for modern web application. It is designed to be a comprehensive starting point for new apps, providing a solid, opinionated foundation with a minimal amount of configuration.

This is a customized version of the original next-forge template.

## Getting Started

Clone the repo using:

```sh
npx better-forge@latest init
```
